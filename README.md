# Análise de dados State of Data Brazil 2022-2023
Utilizando a base de dados [State of Data Brazil 2022-2023](https://www.kaggle.com/datasets/datahackers/state-of-data-2022) extraída do Kaggle com base na pesquisa anual sobre o mercado de trabalho brasileiro de dados feita pela comunidade Data Hackers, foi feita uma análise de dados afim de criar queries que respondam as seguintes perguntas:
1) Qual região tem mais mulheres recebendo a remuneração mais alta segundo a pesquisa?
2)Entre os profissionais com nível superior, quais são as 5 áreas de formação mais comuns?
3) Qual é a representatividade de cada grupo étnico e de gênero em cargos de liderança da área de dados?
4) Qual é a diferença na média salarial entre profissionais de dados com menos de 5 anos de experiência, comparando aqueles com e sem graduação formal?
5) Qual a porcentagem de homens e mulheres em cargos de liderança que não têm formação superior?

### Tecnologias Utilizadas
SQLite
